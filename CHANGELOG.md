# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial project setup with Vite and React for fast development
- Navbar component with responsive design and mobile menu
- Hero section with background image and call-to-action buttons
- Content section with two-column layout for better content presentation
- Testimonial cards section with user reviews and ratings
- Footer with social media links and dynamic copyright year
- Quote page with food quotes from renowned personalities
- Restaurants page with search and filter capabilities
- Foods page with image gallery and category filtering
- Contact page with form validation and Google Maps
- React Router for seamless navigation between pages
- Responsive design for all pages and components
- Search functionality across multiple pages
- Filter functionality for better user experience
- Pagination system for large datasets

### Features
- Mobile-friendly navigation with hamburger menu
- Active link highlighting for better navigation awareness
- Hover effects and animations for enhanced interactivity
- Form validation with user-friendly error messages
- Google Maps integration for location display
- Image galleries with high-quality photos from Unsplash
- Smooth transitions and animations throughout the app
